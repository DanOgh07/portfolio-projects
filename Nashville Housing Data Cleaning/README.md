# Nashville Housing Data Cleaning Project (MySQL)

Standardised and de-duplicated a raw residential property dataset using CTEs, self-joins, and ROW_NUMBER() window functions; applied date normalisation and column restructuring to produce a clean, analysis-ready dataset.

Dataset: https://github.com/AlexTheAnalyst/PortfolioProjects/blob/main/Nashville%20Housing%20Data%20for%20Data%20Cleaning.xlsx
